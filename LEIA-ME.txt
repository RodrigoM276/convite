2. Para publicar, envie todos os arquivos desta pasta para Netlify, Vercel ou GitHub Pages.

PARA PERSONALIZAR

WhatsApp
  Em script.js, a constante whatsappNumber já está com o número correto dos noivos.

Música
  O arquivo musica.mp3 já está na pasta e funciona ao clicar no botão ♪.

Endereço / Mapa
  Para trocar o endereço, edite o link do botão "mapButton" em index.html.

Lista de presentes (Pix)
  Em index.html, procure por id="pixKey" e troque pelo Pix real do casal
  (e-mail, telefone, CPF ou chave aleatória).

Galeria de fotos
  Em index.html, dentro de #galleryGrid, cada item é um placeholder (📷).
  Para usar fotos reais:
    1. Coloque os arquivos de foto nesta mesma pasta (ex.: foto1.jpg, foto2.jpg...)
    2. Troque cada <div class="gallery-item placeholder">...</div>
       por <div class="gallery-item"><img src="foto1.jpg" alt="..."></div>

Mensagens (mural de recados)
  Já funciona: os convidados escrevem uma mensagem e ela é enviada direto para
  o WhatsApp dos noivos, sem necessidade de servidor ou banco de dados.
